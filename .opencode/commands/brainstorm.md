# /brainstorm Command

## Usage

```
/brainstorm [topic or problem]
```

## Examples

```
/brainstorm How might we improve user onboarding?
/brainstorm Ideas for new product features
/brainstorm Ways to optimize database queries
```

## What It Does

This command facilitates structured brainstorming using:

1. **Socratic Questioning** - Asks probing questions to explore the topic deeply
2. **Idea Generation** - Uses techniques like SCAMPER, mind mapping, 5 Whys
3. **Exploration** - Considers multiple angles and perspectives
4. **Synthesis** - Organizes ideas into actionable concepts

## Techniques Used

- **SCAMPER**: Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse
- **5 Whys**: Root cause analysis through repeated questioning
- **Mind Mapping**: Visual organization of ideas
- **What If**: Exploring possibilities

## Output

The agent will:
- Ask clarifying questions
- Generate multiple ideas
- Explore pros/cons
- Suggest next steps
- Document findings

## When to Use

- Starting new projects
- Solving complex problems
- Feature ideation
- Process improvement
- Creative block

## Notes

- No commitment required - explore freely
- All ideas are valid during brainstorming
- Focus on quantity first, quality later
- Build on previous ideas
