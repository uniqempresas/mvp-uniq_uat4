# /ui-ux-pro-max Command

## Usage

```
/ui-ux-pro-max [design request]
```

## Examples

```
/ui-ux-pro-max Design a modern landing page for a fintech app
/ui-ux-pro-max Create a dark mode dashboard with glassmorphism
```

## What It Does

This command invokes the `@frontend-specialist` agent with advanced design capabilities:

1. **Style Selection**
   - Choose from 50+ design styles
   - Modern, minimalist, brutalist, retro, futuristic
   - Industry-appropriate aesthetics

2. **Color Palette**
   - 21 curated color palettes
   - Industry-specific palettes
   - Accessibility-compliant combinations

3. **Typography**
   - 50+ font combinations
   - Google Fonts integration
   - Type scale optimization

4. **Premium Effects**
   - Advanced animations
   - Micro-interactions
   - Visual depth
   - Modern CSS techniques

## Design Styles

- **Brutalist**: Raw, bold, unconventional
- **Glassmorphism**: Translucency, blur effects
- **Neumorphism**: Soft, extruded plastic
- **Minimalist**: Clean, whitespace-focused
- **Retro**: Vintage aesthetics
- **Cyberpunk**: Neon, futuristic
- **Organic**: Natural, flowing shapes

## When to Use

- Premium landing pages
- Marketing sites
- Portfolio websites
- Brand presentations
- High-impact designs

## Output

The agent will:
- Present design options
- Generate component code
- Provide style guide
- Include animation specs
- Ensure responsive design

## Note

This command focuses on visual design excellence and modern trends while maintaining usability and performance.
